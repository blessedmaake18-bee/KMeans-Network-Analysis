K-Means Clustering of Dense Network Topology

How to Run

1. Install Python, NumPy, Pandas, Matplotlib.
2. Run `python code/kmeans.py`.
3. Observe 2D & 3D plots; modify `k` to test different cluster counts.

*Observations:

* Clusters reflect spatial grouping of nodes.
* Critical nodes with low energy are identified.
* Changing `k` affects cluster distribution, highlighting **K-Means limitations** in dense networks.
